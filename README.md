Made with Gemini 3.1 Pro (Antigravity), Claude Sonnet 4.6 (Antigravity), Qwen 3.7-Max (Qoder)


How to run :
Command     	    What it does
npm start	        Starts both servers (recommended)
npm run dev	        Frontend dev server only
npm run server	    Backend API server only
npm run build	    Creates optimized production files in dist/
npm run preview	    Serves the production build locally
For daily development, just run npm start and open http://localhost:5173 in your browser.
